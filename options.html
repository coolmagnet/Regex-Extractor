Regex Patterns

Manage regex patterns to extract text from webpages. Use ( ) to capture the text you want to copy, similar to sed's memory recall (e.g., \1) or grep --only-matching. Only the first ( ) group is copied to the clipboard.

Examples:

  • (.+?) -[^-]*: Captures title text before " - " (e.g., "My Page" from "My Page - Brand").
  • ([a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}): Captures emails (e.g., "user@example.com").
  • <h1>([^<]+)</h1>: Captures h1 content (e.g., "Heading" from "<h1>Heading</h1>").
